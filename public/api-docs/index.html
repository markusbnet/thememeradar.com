📡 Meme Radar — Public API

Free, read-only JSON API for Reddit meme stock trends. No API key required. Rate limit: 60 requests/min per IP.

Base URL: https://thememeradar.com/api/public/v1
All responses follow the shape { "success": true, "data": { ... } } on success and { "success": false, "error": "..." } on failure.

Endpoints

GET/api/public/v1/stocks/trending

Returns the top 10 trending (rising) and top 10 fading (losing interest) stocks, ranked by mention velocity from Reddit.

Query parameters

ParameterTypeDefaultDescription
timeframestring24hWindow size: 1h, 4h, 24h, or 7d

Example request

GET https://thememeradar.com/api/public/v1/stocks/trending?timeframe=24h

Example response

GET/api/public/v1/stocks/:ticker

Returns full details for a single ticker including mention history, evidence posts, and enrichment data.

Example request

GET https://thememeradar.com/api/public/v1/stocks/GME

Example response (abbreviated)

{
  "success": true,

Rate limiting

All public API endpoints share a rate limit of 60 requests per minute per IP address.

Exceeding the limit returns HTTP 429 with body { "success": false, "error": "Rate limit exceeded. Max 60 requests/min." }

Google Sheets — IMPORTJSON recipe

Paste the formula below into any Google Sheet cell to pull live trending data. It requires the free ImportJSON add-on.

=IMPORTJSON("https://thememeradar.com/api/public/v1/stocks/trending?timeframe=24h",
             "/data/trending",
             "noHeaders")

To get only specific columns (ticker + velocity + sentimentCategory):

=IMPORTJSON("https://thememeradar.com/api/public/v1/stocks/trending",
             "/data/trending/ticker,/data/trending/velocity,/data/trending/sentimentCategory",
             "noHeaders")
Tip: Wrap the formula in =IFERROR(…, "Loading…") to display a placeholder while the sheet refreshes. Google Sheets caches external data for ~1 hour by default.

Python / pandas recipe

import requests, pandas as pd

resp = requests.get(
    "https://thememeradar.com/api/public/v1/stocks/trending",
    params={"timeframe": "24h"},
    timeout=10,
)
data = resp.json()["data"]
df = pd.DataFrame(data["trending"])
print(df[["ticker", "mentionCount", "velocity", "sentimentCategory"]])
